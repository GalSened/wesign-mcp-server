üöÄ Getting Started
  1. Run start-with-ngrok.bat to start the server with ngrok tunnel
  2. Copy your ngrok HTTPS URL from the dashboard
  3. Add /mcp to the end of the URL
  4. Go to ChatGPT Settings ‚Üí Connectors
  5. Add custom connector with your URL
  6. Test and enable the connector
  7. Start using beautiful visual prompts!
# Example ChatGPT Prompts "Show me my account dashboard" "Display my documents in a gallery" "Show all templates with their usage" "What's the status of my recent documents?"
üí° Pro Tips
